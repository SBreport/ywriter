# 유튜브 스크립트 & B-Roll — 대본 MD 파일 작성 가이드

## 기본 규칙

| 문법 | 용도 | 예시 |
|------|------|------|
| `# 제목` | 영상 제목 (1개만) | `# 유튜브 시작하기 완벽 가이드` |
| `## 섹션명` | 섹션 구분 (여러 개) | `## 인트로`, `## 본론 1 — 채널 개설` |
| 일반 텍스트 | 대사/나레이션 | 안녕하세요, 오늘은... |

- `#` (H1)은 파일당 **1개만** 사용 → 영상 제목으로 표시됩니다.
- `##` (H2)가 섹션 구분 기준입니다. 각 `##` 블록마다 B-roll 메모칸이 생깁니다.
- `###` 이하, 굵게, 기울임, 리스트 등 마크다운 문법은 자유롭게 사용 가능합니다.

---

## 권장 섹션 구성

```
## 훅 (Hook)
## 인트로
## 본론 1 — [소제목]
## 본론 2 — [소제목]
## 본론 3 — [소제목]
## 아웃트로
## CTA (구독/좋아요)
```

섹션 수는 자유롭게 조절하세요. `##` 뒤에 `—` 로 소제목을 달면 가독성이 좋습니다.

## 예시 대본

```markdown
# 1인 유튜버가 반드시 알아야 할 촬영 팁 5가지

## 훅
"카메라 앞에 서면 할 말이 다 사라지는 분들, 오늘 이 영상 하나로 해결됩니다."

## 인트로
안녕하세요, [채널명]입니다.
오늘은 1인 유튜버가 촬영할 때 꼭 알아야 할 팁 5가지를 정리해봤어요.

## 본론 1 — 조명은 자연광이 답이다
가장 쉽고 효과적인 조명은 **창가 자연광**입니다.

- 창을 정면에 두고 촬영
- 직사광선은 피하고 커튼으로 확산광 만들기
- 촬영 시간대: 오전 10시~오후 2시가 베스트

## 본론 2 — 삼각대 없이 안정적으로 찍는 법
삼각대가 없을 때는 이렇게 해보세요:

1. 책 위에 폰을 세워서 고정
2. 셀카봉을 테이블에 기대기
3. *타이머 촬영*으로 흔들림 방지

## 본론 3 — 음질이 영상의 절반이다
시청자는 화질보다 **음질**에 더 민감합니다.

> 핀마이크 하나만 있어도 퀄리티가 확 달라집니다.

## 본론 4 — B-roll을 미리 계획하세요
편집할 때 가장 막막한 게 "넣을 영상이 없다"는 거예요.
대본 단계에서 어떤 B-roll이 필요한지 적어두면 촬영이 훨씬 수월합니다.

## 본론 5 — 완벽하지 않아도 업로드하자
첫 영상은 무조건 못 찍습니다. 그게 정상이에요.
**10개 찍으면 확실히 달라집니다.**

## 아웃트로
오늘 영상이 도움이 되셨다면 **구독**과 **좋아요** 부탁드립니다.
다음 영상에서는 편집 팁을 다뤄볼게요. 감사합니다!
```

## 사용 방법

1. 위 양식대로 `.md` 파일을 작성합니다.
2. `index.html`을 브라우저에서 열고, MD 파일을 드래그 & 드롭합니다.
3. 각 섹션 옆에 B-roll 태그와 메모를 입력합니다.
4. **HTML 내보내기** 버튼으로 저장합니다.
5. 나중에 내보낸 HTML 파일을 다시 업로드하면 편집을 이어갈 수 있습니다.

### 파일명 규칙
내보내기 시 자동 생성되는 파일명: `YYYYMMDD_영상제목_v번호.html`
- 예: `20260412_촬영팁5가지_v1.html`
- 파일명은 내보내기 전에 수정할 수 있습니다.
