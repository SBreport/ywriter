# youtube_box — 유튜브 작업 도우미 (v3)

1인 유튜버를 위한 **영상 기획 → 원고 작성 → B-roll 계획** 통합 워크플로우 도구.
브라우저에서 바로 사용 가능 (별도 설치 불필요).

## 3단계 워크플로우

각 영상은 **하나의 프로젝트**로 관리되며, 3개 챕터를 순차적으로 진행합니다.

### 1. 기획 (썸네일 / 제목 / 본질)
- **벤치마킹 썸네일 수집**: Ctrl+V 붙여넣기, 드래그, 파일 선택 (최대 12개)
- **내 결정**: 영상 제목, 썸네일 타이틀, 내 썸네일 이미지
- **본질 설정**: 독자(누가 보는가), 목적(왜 만드는가)

### 2. 원고 작성
- 벤치마킹 원고(최대 2개)를 좌측에 띄워두고, 우측에서 내 스타일로 대본 작성
- **도입 30초 자동 강조**: 첫 H2 섹션 또는 "훅" 포함 섹션은 강조 스타일 적용
- 기획 단계에서 정한 **제목 / 썸네일 / 독자 / 목적이 상단에 고정 표시**
- 타임스탬프 자막(`00:00 안녕하세요`) 자동 인식

### 3. B-roll 계획
- 대본은 읽기 전용으로 표시 (수정은 챕터 2로 돌아가서)
- 각 섹션에 **화면녹화 / 별도녹화 / 자료화면** 태그 + 커스텀 태그 + 메모
- **촬영 체크리스트** 자동 생성 → 클립보드 복사해서 현장에서 사용

## 데이터 저장

모든 작업은 **브라우저 내부 저장소(localStorage + IndexedDB)**에 자동 저장됩니다.

- **ZIP 백업**: 프로젝트를 단일 ZIP 파일로 내보내기 (텍스트 + 이미지 모두 포함)
- **ZIP 불러오기**: 다른 PC에서도 ZIP 파일을 드롭하면 작업 이어가기 가능
- **전체 백업**: 모든 프로젝트를 한 번에 ZIP으로 묶기

## 주요 기능

- 다크/라이트 테마 토글
- 자동 저장 (작업 변경 시)
- 3단계 잠금 시스템 (이전 단계를 완료해야 다음 단계 진입)
- 상단 고정 컨텍스트 바 (본질을 잃지 않게)
- 마크다운 기반 대본 작성 (H1/H2/H3 계층)
- 드래그 리사이즈 컬럼 비율

## 단축키
- `Ctrl+S` 수동 저장
- `Ctrl+E` ZIP 내보내기
- `Ctrl+1/2/3` 섹션 레벨 변경 (제목 입력 중)
- `Ctrl+V` 이미지 붙여넣기 (챕터 1)
- `Esc` 모달 닫기
