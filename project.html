프로젝트 가이드

3단계 워크플로우

  1. 기획: 벤치마킹 썸네일을 수집하고, 영상 제목 · 썸네일 타이틀 · 독자 · 목적을 정합니다. 이 4가지는 다음 단계에서 항상 상단에 표시됩니다.
  2. 원고 작성: 벤치마킹 원고를 참고하며 내 스타일로 대본을 씁니다. 첫 H2 섹션은 "도입 30초"로 자동 강조됩니다.
  3. B-roll: 대본은 읽기 전용으로 표시되고, 각 섹션에 촬영 태그와 메모를 추가합니다.

단축키

  • Ctrl+S 수동 저장 | Ctrl+E ZIP 내보내기 | Esc 모달 닫기
  • Ctrl+V 기획 단계에서 이미지 붙여넣기
  • Ctrl+1/2/3 섹션 제목에서 H1/H2/H3 변경

데이터 저장

모든 변경은 자동 저장됩니다. ZIP 백업 버튼으로 프로젝트를 파일로 내보낼 수 있습니다.

YouTube API 키 설정

YouTube 영상 URL을 붙여넣으면 썸네일 · 제목 · 채널명 · 업로드일 · 영상 길이를 자동으로 가져옵니다.

API 키는 브라우저에만 저장되며, 외부로 전송되지 않습니다.

API 키 발급 방법 (약 3분)

  1. 구글 클라우드 콘솔 접속 → 새 프로젝트 만들기
  2. "APIs & Services" → "Library" → YouTube Data API v3 검색 → 사용 설정
  3. "Credentials" → "Create Credentials" → "API key"
  4. 생성된 키를 아래에 붙여넣기

하루 10,000회 무료 쿼리 (개인 용도로 충분합니다). API 키는 HTTP 리퍼러 제한을 거는 것을 권장합니다.
