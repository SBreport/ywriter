ywriter 사용 가이드

워크플로우

하나의 영상 프로젝트는 3개 챕터로 구성됩니다.

  • 1. 기획: 벤치마킹 썸네일 수집 → 내 영상 제목 / 썸네일 타이틀 / 독자 / 목적 확정
  • 2. 원고 작성: 벤치마킹을 참고하며 내 스타일로 대본 작성 (도입 30초 강조)
  • 3. B-roll: 각 섹션에 촬영 태그/메모 추가

1단계에서 정한 정보가 2, 3단계 상단에 항상 고정 표시되어 본질을 잃지 않게 합니다.

데이터 저장

  • 모든 작업은 브라우저 내부(IndexedDB + localStorage)에 자동 저장됩니다.
  • ZIP 내보내기로 프로젝트를 백업하고, 다른 PC에서 ZIP 불러오기로 이어 작업할 수 있습니다.
  • 이미지(썸네일)는 붙여넣기(Ctrl+V), 드래그 앤 드롭, 파일 선택 모두 지원합니다.

단축키

  • Ctrl+S 임시저장 | Ctrl+Z 되돌리기
  • Ctrl+E ZIP 내보내기 | Ctrl+P 대본 모드(텔레프롬프터)
  • Ctrl+1/2/3 제목에서 섹션 레벨 변경
